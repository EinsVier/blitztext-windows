# BlitzText Windows 0.5.0

BlitzText 0.5.0 simplifies text workflows and makes prompts easier to understand, customize, and reuse.

## Highlights

- Simplifies the visible workflows to Transcribe, Improve, and Calm down.
- Adds a reusable emoji option to Improve, Calm down, and result reprocessing.
- Migrates the previous standalone emoji workflow automatically.
- Separates the emoji instruction from the main workflow prompts.
- Reworks all 14 prompt presets in German and English for clearer, shorter, and more reliable output.
- Strengthens prompt handling for facts, names, dates, commands, code, uncertainty, and sensitive text.
- Improves emoji behavior with explicit handling for formal, technical, legal, medical, conflict-related, and sensitive content.
- Renames the Backup tab to Info and adds links to GitHub, issue reporting, and the MIT license.
- Groups settings import and export under a clearer settings backup section.

The MSI requires the .NET 8 Desktop Runtime.
