# README demo recording

Record a real 10-15 second workflow for the README:

1. Open a neutral text editor beside BlitzText.
2. Start with an empty document and no private content visible.
3. Press the Improve hotkey.
4. Dictate one short, natural sentence.
5. Stop recording and show the polished text appearing in the editor.
6. End after the result is visible for about two seconds.

Capture requirements:

- 1280x720 or smaller crop focused on the editor and BlitzText status.
- No API keys, local addresses, personal names, notifications, or unrelated windows.
- Use the real application and provider response; do not simulate the result.
- Prefer MP4 as the source and export an optimized GIF or WebM for the README.
- Keep the final asset short, silent, and small enough to load quickly on GitHub.
